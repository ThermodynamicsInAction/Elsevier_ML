# Elsevier_ML
Codes and source files for the application of predictive methods to estimate density and speed of sound in ionic liquids

The following files and repositories contain codes for predicting the speed of sound/density in ionic liquids. The project includes code for: spinodal method, neural network and a number of machine learning methods.
